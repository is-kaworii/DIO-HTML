<h1> até <h6> Headings

<h1> até <h6> são tags utilizadas para definir títulos Headings (cabeçalhos).

Sendo <h1> com maior importancia e <h6> com menor importância semântica.

Geralmente é utilizado apenas uma <h1> por página, para representar o título principal da página toda.

<p> Paragraph

A tag <p> define um parágrafo.

<mark> Mark

A tag <mark> define textos para serem marcados ou destacados.

<small> Small

A tag <small> define um texto melhor (como copyright e outros side-comments).

<i> Itallic

A tag <i> define parte do texto com um intensão diferente. O conteúdo geralmente fica formatado em itálico.

A tag <i> geralmente é utilizado para indicar um termo, uma frase em outro idioma, um pensamento, um nome extrangeiro, etc.

Apenas use <i> quando não à um elemento semantico mais apropriado, como:

  • <em> (enfase no texto)
  • <strong> (texto importante)
  • <mark> (marcar ou destacar texto)
  • <cite> (o título de um trabalho)
  • <dfn> (uma definição de um termo)

<u> Underline

A tag <u> representa um texto é articulado e tem um estilo diferente do texto normal, como palavras com erros ortográficos. O conteúdo interno é normalmente exibido com um sublinhado.

Dica: Evite usar o elemento <u> onde ele pode ser confundido com um hiperlink!

<strong> Strong

A tag <strong> é usada para definir texto com forte importância. O conteúdo dentro é normalmente exibido em negrito.

Dica: Use a tag <b> para especificar texto em negrito sem nenhuma importância extra!

<ol> ordered list

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

A tag <li>; é usada para definir cada item da lista.

Dica: Use CSS para estilizar listas.

Dica: Para listas não ordenadas, use a tag <ul>.

<ul> unordered list

A tag <ul> define uma lista não ordenada (com marcadores).

Use a tag <ul> junto com a tag <li>; para criar listas não ordenadas.

Dica: Use CSS para estilizar listas.

Dica: Para listas ordenadas, use a tag <ol>.

<li> list item

A tag <li> define um item de lista.

A tag <li> é usada dentro de listas ordenadas (<ol>), listas não ordenadas (<ul>) e em listas de menu (<menu>).

Em <ul> e <menu>, os itens de lista geralmente serão exibidos com marcadores.

Em <ol>, os itens de lista geralmente serão exibidos com números ou letras.

Dica: use CSS para estilizar listas.

<a> ancor point

A tag <a> define um hiperlink, que é usado para vincular de uma página para outra.

O atributo mais importante do elemento <a> é o atributo href, que indica o destino do link.

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

  • Um link não visitado é sublinhado e azul
  • Um link visitado é sublinhado e roxo
  • Um link ativo é sublinhado e vermelho

<hr>

A tag <hr> define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).

O elemento <hr> é mais frequentemente exibido como uma régua horizontal que é usada para separar conteúdo (ou definir uma mudança) em uma página HTML.

<sub> Sub

A tag <sub> define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.

Dica: Use a tag <sup> para definir texto sobrescrito.

<sup> Sup

A tag <sup> define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW.

Dica: Use a tag <sub> para definir texto subscrito.

<blockquote> Block Quote

A tag <blockquote> especifica uma seção que é citada de outra fonte.

Os navegadores geralmente recuam elementos <blockquote> (veja o exemplo abaixo para ver como remover o recuo).

Dica: Use <q> para citações inline (curtas).
